카카오 로그인api 및 톡캘린더api를 활용한 카카오톡 친구 생일 모아보기 서비스


1. 기획의도
일상생활을 바쁘게 하다보면 기억해야 할 것들을 깜박하고 잊는 경우가 많습니다.
그 중에서도 '소중한 사람들의 생일'을 잊지 않고 축하해주기 위해서 해당 서비스를 기획했습니다.
또한, 오픈 api를 직접 활용해서 개발해보고자 카카오 로그인api 및 톡캘린더api를 선택했습니다.

2. 개발 기간
2023.12.28 ~ 현재 (진행중)

3. 사용한 기술스택
JAVA 17
Spring Boot
MyBatis 3.0.2
MySQL
Gradle 8.3
Thymeleaf(html)
javascript
